Front End

I searched for information about front end. I took notes about the following video(s) Front-End Development on YouTube

  • HTML is used to display content on a webpage and is primarily used in front-end development.
  • CSS is used to style web pages and make them visually appealing.
  • JavaScript makes web pages interactive and dynamic.
  • DOM (Document Object Model) is a feature that allows JavaScript to manipulate the webpage's structure.
  • JavaScript frameworks like React, Angular, and Vue.js are often used to simplify front-end development.
  • Tools like Webpack are used to bundle JavaScript code, making it more efficient for deployment.
  • TypeScript is a popular transpiler used with JavaScript for adding static typing.
  • CSS uses a preprocessor like Sass, which allows you to split CSS into different files for better organization.
  • Frameworks like Bootstrap provide pre-designed CSS components to streamline front-end development.
  • For sending data from the front end to the backend, JavaScript uses methods like XMLHttpRequest, Axios, and the Fetch API.
